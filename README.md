支持：
- 按车次号查询历史担当
- 按车组号查询历史担当
- 查看车站时刻表与参考车型
- 查看列车时刻表
- 查看列车交路表
- 预测列车未来担当
- 收藏车次、车组
- 订阅车次、车组的运用情况
- 一键导出历史数据

开放 API，可供其他网站、应用和 AI 接入。
